Declaración de conformidad con las Normas de Accesibilidad Web (WCAG)

Yo, Marcos Cobo, declaro que el sitio web TravellRoulette cumple con las pautas establecidas en las Normas de Accesibilidad Web (WCAG) en su versión 2.2, emitidas por el Consorcio World Wide Web (W3C). Este documento tiene como objetivo certificar que el sitio web TravellRoulette cumple con los siguientes criterios de accesibilidad:

  • Accesibilidad

    TravellRoulette está haciendo sus mayores esfuerzos para facilitar el acceso a las personas con discapacidad, trabajando por una web accesible desde cualquier dispositivo o sistema operativo utilizado. Actualmente la configuración de los navegadores, y las soluciones de los diferentes sistemas operativos permiten navegar de una forma más fácil en nuestra Web. El compromiso de TravellRoulette es seguir trabajando en herramientas y soluciones de accesibilidad.
